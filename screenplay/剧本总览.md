# 场景设计要点

## 核心场景

### 1. 北京出租屋
- 40 平米，老旧
- 一张床、一张桌子、一个衣柜
- 窗台上有几本书、落灰的跑步机
- 关键道具：相框、绿萝、马克杯

### 2. 中关村写字楼
- 25 层，格子间如牢笼
- 孤灯、应急灯、惨白光斑
- 象征：社畜的牢笼

### 3. 星际联邦分部
- 普通写字楼外表
- 内部现代简洁
- 训练馆有 5 米高金属测试柱
- 圆形擂台带透明防护罩

### 4. 月球基地（后续）
- 科幻感、金属质感
- 环形结构
- 训练设施先进

# 开发笔记

## 改编策略

1. **前三集节奏**：慢热开篇建立共情，第 3 集开始加速
2. **视觉化优先**：将内心独白转化为视觉动作
3. **人物关系**：提前铺垫苏雨晴与林默的感情线
4. **悬念设置**：每集结尾设置钩子

## 潜在挑战

1. **修真元素视觉化**：需要用 CGI 呈现灵力、功法等
2. **世界观信息量**：需要平衡 exposition 与叙事节奏
3. **预算考虑**：月球基地、星际场景成本较高

## 建议

- 第 1 季聚焦地球 + 月球基地，控制场景规模
- 炎帝记忆用风格化画面（剪影/抽象）降低 CGI 成本
- 起源系统用 UI 特效呈现，成本可控
