# 第 105 章 新的征程

火星，边境星域基地。

林默回到基地后，开始闭关修炼。

"宿主，你……"

"我知道。"林默说，"我需要……时间。"

【好。】

林默盘膝而坐，开始运转《万族神诀》。

金色的光芒在他周身流转。

"林默，你……"苏雨晴守在门外。

"别打扰我。"林默说，"我……在冲击瓶颈。"

"……好。"

苏雨晴点点头。

时间一天天过去。

林默的气息，越来越强。

"宿主，你……"

"我知道。"林默在心里说，"我感觉到了，瓶颈在松动。"

【好。】

突然，林默身上爆发出一股强大的气息。

"突破了！"苏雨晴惊喜道。

职业者 22 级！

林默睁开眼睛，眼中金光闪烁。

"感觉……怎么样？"苏雨晴问。

"很好。"林默说，"非常好。"

"我能感觉到，我的力量，比之前强了十倍不止。"

"十倍？"

"对。"林默说，"职业者 22 级，是一个新的境界。"

"那……你现在能打过谁？"

"不知道。"林默笑了，"但我想……域外天魔皇，应该有一战之力。"

"……好。"

苏雨晴靠在他怀里。

"雨晴，谢谢你。"

"谢什么。"苏雨晴说，"我是你妻子。"

"对。"林默笑了，"你是我妻子。"

"我们会一直在一起。"

"嗯。"

两人相拥而立。

"宿主，有……情况。"

"什么？"

"星际联邦……发来……邀请。"

"邀请？"

"对。"起源说，"他们……想……见你。"

"见我？"林默愣了一下，"为什么？"

"不知道。"起源说，"但……应该……是好事。"

"……好。"

林默站起身。

"各位，我们……出发。"

"是！"

所有人登上飞船。

"雨晴，你……"

"我跟你一起去。"苏雨晴说。

"……好。"

林默和苏雨晴，登上旗舰。

"铁柱，晓芸，子轩，你们……跟我一起。"

"是！"

飞船起飞，驶向星际联邦总部。

一天后，星际联邦总部。

"到了。"林默说，"下船。"

众人下船。

"林默主席，欢迎欢迎。"一个中年人迎了上来。

"你是……"

"我？"中年人说，"我是……星际联邦，主席。"

"星际联邦主席？"林默瞳孔一缩。

"对。"主席说，"我……是来……感谢你的。"

"感谢我？"

"对。"主席说，"你……灭了域外天魔将军，使者。"

"这……是我们……人类联盟的……荣耀。"

"……好。"

林默点点头。

"那……你……找我……有什么事？"

"事？"主席愣了一下，随即大笑，"哈哈，没什么大事。"

"就是……想……见见你。"

"见我？"

"对。"主席说，"你……是人类联盟的……领袖。"

"我……是星际联邦的……主席。"

"我们……应该……合作。"

"……好。"林默说，"我……同意。"

"那……我们……签订……协议。"

"……好。"

林默和主席，签订协议。

"林默，你……"主席笑了，"以后……我们……就是盟友了。"

"对。"林默说，"我们……是盟友。"

"那……我……有一个……消息……告诉你。"

"什么消息？"

"你……想知道……域外天魔的……来历吗？"

"域外天魔的来历？"林默瞳孔一缩。

"对。"主席说，"他们……不是……这个宇宙的。"

"不是这个宇宙的？"

"对。"主席说，"他们……来自……暗宇宙。"

"暗宇宙？"

"对。"主席说，"那是……一个……高维空间。"

"高维空间？"

"对。"主席说，"那里……有很多……强大的存在。"

"域外天魔皇……只是……其中一个。"

"……什么？！"

林默震惊了。

"那……我……"

"你……不用怕。"主席说，"你……有炎帝的传承。"

"炎帝……是……这个宇宙……最强的存在。"

"只要……你……继承了他的……全部传承。"

"你……就可以……击败……域外天魔皇。"

"……好。"

林默点点头。

"那……我……需要……做什么？"

"你……需要……找到……剩下的……起源碎片。"

"起源碎片？"

"对。"主席说，"起源……是炎帝留下的……唯一遗物。"

"只要……你……收集了……所有的……起源碎片。"

"你……就可以……继承……炎帝的……全部传承。"

"……好。"

林默记住了。

"那……起源碎片……在哪？"

"不知道。"主席说，"但……我……可以……帮你……寻找。"

"……谢谢。"

"不用谢。"主席说，"我们……是盟友。"

"……好。"

林默和主席，告别。

"宿主，你……"

"我知道。"林默在心里说，"我……需要……找到……起源碎片。"

【对。】

"林默，你……"苏雨晴走过来。

"我没事。"林默说，"我们……回家。"

"……好。"

林默和苏雨晴，登上飞船。

"铁柱，晓芸，子轩，我们……回家。"

"是！"

飞船起飞，驶向火星。

而此刻，宇宙深处。

"林默……和星际联邦……合作了。"

"知道了。"

"那我们……"

"不用怕。"那声音说，"林默……是个威胁。"

"但……我……不怕他。"

"……是！"

黑暗中，几双眼睛闪烁着寒光。

一场大战，即将爆发。

林默站在飞船舷窗前，望着无尽的星空。

「暗宇宙……高维空间……无论你们在哪里，我都会找到你们。」

「炎帝的传承，不会在黑暗中熄灭。」

苏雨晴走到他身边，轻轻握住他的手。

「在想什么？」

「在想我们的未来。」林默转过头，眼中闪烁着坚定的光芒，「雨晴，相信我，我会让人类在这个宇宙中，拥有属于自己的位置。」

「我一直相信你。」苏雨晴靠在他肩头。

飞船驶向火星，消失在浩瀚星海中。

【第 105 章 完】
