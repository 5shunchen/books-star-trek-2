# 《星塔之心》- 2150科幻冒险小说

> 一个八岁男孩、一座通天星塔、一颗跨越万年的初心。

---

## 一、故事简介

2150年，人类迈入"星际文明萌芽期"。在新天穹城——一座以星塔为中心的超级空中立体都市——八岁男孩**小光**骑着他的蓝色虎鲨飞行器，意外被远古星塔选中，成为第1001位"星际少年先锋官"。

**类型**：温暖科幻 / 儿童文学 / 长篇连载（约200章，10卷）
**基调**：明亮的科技乌托邦，充满希望与探索精神
**核心主题**：善意不是一种选择，而是一种本能——就像一个孩子看到星空时脱口而出的那句"太棒了"

---

## 二、世界观设定

### 1. 新天穹城结构（由下至上）

| 层级 | 名称 | 功能 |
|------|------|------|
| L1 | 地面层 | 保留旧时代能源设施，基石工程师驻守，维护核电站和无线充电基站 |
| L2 | 基础设施层 | 能源分配、水处理、废物循环 |
| L3 | 居民生活层 | 住宅区、社区、学校，小光家所在的"云杉社区" |
| L4 | 商业文娱层 | 空中游乐场、全息影院、购物中心 |
| L5 | 行政科研层 | 能源管理局、星塔运维中心 |
| L6+ | 星塔区域 | 星塔基座、接口塔、能量分配器 |
| 顶端 | 星塔之心 | 艾尔萨文明遗留的"种子"，地球文明的守护者 |

### 2. 能源体系

| 能源 | 来源 | 用途 | 覆盖范围 |
|------|------|------|----------|
| **星能** | 星塔之心（艾尔萨文明的种子） | 全城无线供电：飞行器、全息设备、食物打印机、磁悬浮床等 | 地球表面及近地轨道（约50万公里） |
| **核聚变** | 聚变反应堆 | 深空探测器、星际飞船 | 全太阳系 |
| **核能（遗产）** | 地面层旧时代核电站 | 地面层维护设备、L1基础设施 | 仅L1地面层 |
| **太阳能** | 卫星太阳能板 | 地面层辅助供电 | 仅L1地面层 |

### 3. 星塔来历

- **五千年**前：艾尔萨文明的种子飞船坠落东亚大陆，在地下缓慢生长
- **两千年**前：星塔外壳演化完成，开始从休眠中缓慢苏醒
- **2105年**：人类科学家在勘探地热时发现星塔
- **2118年**：星能首次被捕获并应用，新天穹城建立
- **2142年**：小光出生，星塔之心发出五千年最强脉动——"第1001次睁眼"

### 4. 艾尔萨文明

- **一万年前**：艾尔萨文明达到终极形态，意识融入宇宙信息场，"毕业"了
- 临终前：将文明精华装入无数"种子飞船"，散布到银河系每个角落
- **使命**：等待当地文明成长到足以理解"善意"的那一天
- **地球的特殊性**：唯一一个在未被主动接触的情况下，依然本能地说出"太棒了"的文明

### 5. 核心科技

| 科技 | 说明 |
|------|------|
| **星能广播** | 星塔之心 → 能量光环 → 无线传输 → 全球覆盖。不是电磁波，而是从量子真空涨落中提取能量的共振器 |
| **反重力技术** | 用于训练场、飞行器、磁悬浮设备。可调节重力系数（0.05G ~ 2G） |
| **全息投影** | 全息通讯、全息娱乐、全息教学。设备为薄如纸的面板，可折叠 |
| **飞行器** | 民用版最高时速120km/h（儿童款），专业版可达300km/h。靠星能驱动，矢量推进 |
| **细胞培育肉** | 不需要屠宰动物，从细胞直接培育出肉类。口感和真肉一致 |
| **基因复苏** | 复活了侏罗纪蜂鸟、迷你猛犸象等已灭绝物种，用于生态研究和观赏 |
| **AI助手** | 每个家庭配有小型AI（如小葵），全息投影形象，管理家庭日常 |
| **冥想共鸣** | 通过心跳同步与星塔之心建立意识连接，极少数人拥有"高频共振体质" |

## 四、分卷大纲（10卷 × 约200章）

### 第一卷：空中少年的日常（第1-20章）✅

- **关键人物**：小光、阿铁、小樱、林老师、爸爸妈妈
- **情感基调**：温暖、明亮、充满童趣
- **主线关联**：铺垫星塔之心的存在，埋下"高频共振体质"的伏笔

### 第二卷：星塔之光（第21-40章）✅
- **核心事件**：星塔正式选中小光，林老师身份揭秘，妈妈的秘密揭晓，数据收集者事件
- **关键人物**：赵工、沈明薇、数据收集者组织
- **情感基调**：神秘感递增，危机与温暖并存
- **主线关联**：小光首次进入星塔内部，与星塔之心建立深度共鸣

- **情感基调**：史诗感、探索未知的勇气

## 五、故事核心主题

1. **善意是一种本能** —— 不是经过计算的道德选择，而是看到美好事物时脱口而出的"太棒了"
2. **科技应该有温度** —— 星能的暖金色不是偶然，它是每一次人类心跳的回响
3. **孩子比大人更接近真理** —— 成年人会计算利弊、恐惧后果，而孩子只看得到"美"
4. **理解不是前提，体验才是** —— 星塔之心不要求你"懂"它，它只要求你"感受"它
5. **每一个选择都在投票** —— 你的每一个"太棒了"或"有什么用"，都在塑造未来的样子

## 六、视觉风格参考

- **整体色调**：暖金色（星能）、深蓝色（夜空）、蓝绿色（全息光）
- **飞行器设计**：流线型、仿生学（鲨鱼、蝴蝶、蜻蜓）
- **星塔**：白色半透明塔身，内部有流动的光脉，顶端金色光环旋转
- **城市风格**：立体化、通透感、大量使用全息和磁悬浮技术
- **星塔之心**：暖橙色液态金属晶体，约两米高，表面流动如水面

## 七、写作规范

| 规范 | 要求 |
|------|------|
| **每章字数** | 3000-5000字（纯正文，不含标题） |
| **章节命名** | 每章有独立标题，如"蜂鸟叫早服务"、"星塔之光" |
| **推送方式** | 每5章调用一次 `push_chapters.py` 推送到 `books-001` 分支 |
| **进度汇报** | 每20章向用户做一次详细汇报（含字数统计、剧情总结） |
| **语言风格** | 面向8-14岁读者，温暖明快，避免晦涩术语，科幻元素通过角色体验自然呈现 |
| **章节结尾** | 每章结尾标注"第X章完" |
| **Agent日志** | 推送时打印步骤和结果，确保可验证 |

## 十、如何续写（给未来AI助手的指南）

如果你在任何AI平台上接手这个故事，请按以下步骤开始：

1. **读 README**：你已经在这里了 ✅
2. **拉取仓库**：`git clone https://github.com/5shunchen/books-star-trek-2.git && git checkout main`
3. **读已有章节**：从第1章开始快速浏览，把握人物性格、对话风格、世界观细节
4. **确认当前进度**：查看 `chapters/` 目录中已有的最高章节号，并阅读 `plan.md` 中的版本规划
5. **继续写作**：按照 `prompt.md` 的编写流程与分卷大纲的规划，延续上一章的剧情，保持人物一致性和风格统一
6. **提交**：遵循 Conventional Commits 规范，建议每个章节单独提交，每完成一个版本创建 Git Tag 并推送
7. **汇报**：每20章向用户做一次详细进度汇报
