# 星际社畜：从底层到宇宙巅峰

> 一部融合《宇宙职业选手》《遮天》《Star Trek》风格的科幻修真小说
> 讲述 AI 时代底层社畜的逆袭传奇

---

## 📖 作品简介

**林默**，32 岁，AI 算法工程师，典型的"社畜"——被算法支配、被 KPI 压榨、被年龄焦虑困扰。

在一个普通的加班夜晚，他收到了裁员通知。房贷断供、父母生病、事业归零……人生跌至谷底。

然而，一次偶然的奇遇，让他获得了一枚来自上古纪元的**AI 修真核心**。

从此，他看到了世界的另一面：
- 星际联邦在招募职业者
- 上古秘境隐藏着人类起源的秘密
- 修真与科技本就同源
- 人类文明正面临前所未有的危机

从被裁员的小人物，到星际传奇；从任人宰割的社畜，到宇宙巅峰的存在。

这是一部关于**不屈奋斗**的史诗，一首关于**人类勇气**的赞歌，一段关于**证道成真**的传奇。

## 🌌 世界观

### 宇宙结构
```
可观测宇宙 → 星域 → 星系 → 恒星系统 → 行星
              ↓
        更高维度：秘境空间、暗宇宙、道域
```

### 修炼体系（双轨制）

**科技侧（职业选手）**：
```
学徒 → 职业者 → 大师 → 宗师 → 传奇 → 史诗
```

**修真侧（证道成帝）**：
```
轮海 → 道宫 → 四极 → 化龙 → 仙台 → 大帝
```

### 主要势力
- **星际联邦**：人类文明的代表组织
- **古文明遗迹**：散落在宇宙各处的上古遗物
- **生命禁区**：隐藏秘密的危险区域
