# 小说规划文档

## 小说基本信息

**书名**：《星际社畜：从底层到宇宙巅峰》

**类型**：科幻修真 / 星际冒险 / 职场逆袭

**目标字数**：250~300 万字

**连载状态**：连载中

---

## 核心卖点

1. **AI 时代社畜共鸣**：主角是典型的 AI 时代底层程序员，被算法支配、被 KPI 压榨、被年龄焦虑困扰，让读者产生强烈代入感
2. **三重世界观融合**：
   - 《宇宙职业选手》的职业竞技体系
   - 《遮天》的宏大修真宇宙观
   - 《Star Trek》的星际探索精神
3. **爽点密集**：从被裁员的绝境到获得上古传承，从被看不起的小人物到宇宙巅峰的存在
4. **正向价值观**：不屈不挠的奋斗精神，对公平正义的追求，对人类文明的责任感

---

## 世界观设定

### 一、宇宙观（融合《遮天》+《Star Trek》）

#### 宇宙结构
```
可观测宇宙 → 星域 → 星系 → 恒星系统 → 行星

更高维度：
- 秘境空间（遮天式古遗迹）
- 暗宇宙（高维文明栖息地）
- 道域（修真者证道之地）
```

#### 宇宙历史纪元
1. **太古纪元**：先天神魔时代，文明起源
2. **上古纪元**：修真文明鼎盛，万族争霸
3. **中古纪元**：科技崛起，修真衰落
4. **近古纪元**：AI 诞生，人机共存
5. **当前纪元**：星际大航海前夜

#### 关键宇宙设定
- **星路**：连接各星系的古代通道，类似遮天的"星路"
- **生命禁区**：隐藏着上古秘密的危险区域
- **星际联邦**：人类文明的代表组织，类似 Star Trek 的 Federation
- **古文明遗迹**：散落在宇宙各处的上古文明遗物

### 二、修炼/职业体系（融合《宇宙职业选手》+《遮天》）

#### 双轨体系
```
科技侧（职业选手体系）：
学徒 → 职业者 → 大师 → 宗师 → 传奇 → 史诗

修真侧（遮天证道体系）：
轮海 → 道宫 → 四极 → 化龙 → 仙台 → 大帝
```

#### 主角的独特路径
- 以 AI 技术为根基，融合修真传承
- 在星际联邦的职业体系中攀升
- 在古遗迹中觉醒上古血脉

### 三、星球观

#### 主要星球类型
1. **主星（地球/人类母星）**：AI 高度发达，社畜文化盛行
2. **修真星**：保留古法传承的封闭星球
3. **科技星**：纯粹科技发展，AI 治理
4. **资源星**：星际 mining 基地，底层劳工聚集地
5. **边境星**：人类文明边缘，危险与机遇并存

### 四、故事观

#### 核心主题
- **个人成长**：从底层社畜到宇宙巅峰
- **文明责任**：个人命运与人类文明绑定的使命感
- **人性光辉**：在绝境中依然保持善良和正义
- **探索精神**：对未知宇宙的好奇与敬畏

#### 叙事节奏
- 每 3-5 章一个小高潮
- 每 20-30 章一个大高潮
- 每卷一个完整的起承转合

## 人物设定

### 主角：林默

#### 基本信息
- **年龄**：32 岁
- **职业**：AI 算法工程师（被裁员边缘）
- **出身**：普通家庭，小镇做题家
- **性格特点**：
  - 坚韧不拔：再难也不放弃
  - 正直善良：有底线，不坑人
  - 隐忍内敛：不善于表达情绪
  - 聪明机智：善于用智慧解决问题
  - 不服输：越挫越勇的性格

#### 外貌特征
- 身高 178cm，偏瘦
- 黑眼圈明显（长期加班）
- 眼神坚毅（核心特征）
- 穿着简单（程序员标配）

#### 成长弧光
```
被裁员 → 获得奇遇 → 进入星际联邦 → 发现上古秘密 
→ 卷入星际纷争 → 成长为传奇 → 证道宇宙巅峰
```

#### 金手指/奇遇
- **上古 AI 核心**：融合修真与科技的奇异存在
- **功能**：
  - 推演功法（科技侧）
  - 解析遗迹（修真侧）
  - 优化修炼（双轨融合）

### 重要配角

#### 1. 苏雨晴（女主）
- **身份**：星际联邦年轻军官
- **性格**：果敢、正义、有领导力
- **与主角关系**：从互相利用到生死与共

#### 2. 老陈（导师角色）
- **身份**：主角前公司技术总监，隐退大佬
- **性格**：表面散漫，实则深谋远虑
- **作用**：主角的引路人

#### 3. 王胖子（死党）
- **身份**：主角大学同学，自媒体博主
- **性格**：乐观、仗义、话多
- **作用**：提供情报、调节气氛

#### 4. 反派：赵天成
- **身份**：主角前公司高管，后投靠外星势力
- **性格**：自私、贪婪、不择手段
- **与主角关系**：多次陷害，最终对决

## 第一卷详细大纲（0.1.0 - 0.5.0）

### 开篇章节设计（0.1.0 版本）

#### 第 1 章：被裁员的夜晚
- **场景**：深夜办公楼，主角最后一个离开
- **冲突**：收到裁员通知，房贷压力，父母生病
- **悬念**：回家路上遇到神秘事件
- **情绪**：压抑、绝望中的一丝不甘

#### 第 2 章：神秘遗物
- **场景**：回家路上的小巷，发现神秘物体
- **冲突**：神秘物体与主角产生共鸣
- **悬念**：物体到底是什么
- **情绪**：好奇、震惊

#### 第 3 章：觉醒
- **场景**：主角出租屋
- **冲突**：AI 核心与主角融合，身体改造
- **悬念**：融合后的变化
- **情绪**：痛苦后的新生

#### 第 4 章：真相一角
- **场景**：AI 核心记忆库
- **冲突**：获取部分上古信息，世界观颠覆
- **悬念**：更大的世界在等待
- **情绪**：震撼、决心

#### 第 5 章：新的开始
- **场景**：主角重新振作，制定计划
- **冲突**：如何利用新能力改变命运
- **悬念**：星际联邦的邀请
- **情绪**：希望、期待

## 创作注意事项

1. **人物塑造**：主角性格必须始终如一，成长要有铺垫
2. **场景描写**：每个场景都要有画面感，让读者身临其境
3. **节奏把控**：张弛有度，高潮后要有缓冲
4. **逻辑自洽**：设定不能前后矛盾
5. **情感共鸣**：社畜困境要真实，逆袭要爽快
6. **留白艺术**：不要把所有话说尽，给读者想象空间
