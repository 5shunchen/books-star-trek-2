# 小说规划文档

## 小说基本信息

**书名**：《星际社畜：从底层到宇宙巅峰》

**类型**：科幻修真 / 星际冒险 / 职场逆袭

**目标字数**：250~300 万字

**连载状态**：连载中

---

## 核心卖点

1. **AI 时代社畜共鸣**：主角是典型的 AI 时代底层程序员，被算法支配、被 KPI 压榨、被年龄焦虑困扰，让读者产生强烈代入感
2. **三重世界观融合**：
   - 《宇宙职业选手》的职业竞技体系
   - 《遮天》的宏大修真宇宙观
   - 《Star Trek》的星际探索精神
3. **爽点密集**：从被裁员的绝境到获得上古传承，从被看不起的小人物到宇宙巅峰的存在
4. **正向价值观**：不屈不挠的奋斗精神，对公平正义的追求，对人类文明的责任感

---

## 世界观设定

### 一、宇宙观（融合《遮天》+《Star Trek》）

#### 宇宙结构
```
可观测宇宙 → 星域 → 星系 → 恒星系统 → 行星

更高维度：
- 秘境空间（遮天式古遗迹）
- 暗宇宙（高维文明栖息地）
- 道域（修真者证道之地）
```

#### 宇宙历史纪元
1. **太古纪元**：先天神魔时代，文明起源
2. **上古纪元**：修真文明鼎盛，万族争霸
3. **中古纪元**：科技崛起，修真衰落
4. **近古纪元**：AI 诞生，人机共存
5. **当前纪元**：星际大航海前夜

#### 关键宇宙设定
- **星路**：连接各星系的古代通道，类似遮天的"星路"
- **生命禁区**：隐藏着上古秘密的危险区域
- **星际联邦**：人类文明的代表组织，类似 Star Trek 的 Federation
- **古文明遗迹**：散落在宇宙各处的上古文明遗物

### 二、修炼/职业体系（融合《宇宙职业选手》+《遮天》）

#### 双轨体系
```
科技侧（职业选手体系）：
学徒 → 职业者 → 大师 → 宗师 → 传奇 → 史诗

修真侧（遮天证道体系）：
轮海 → 道宫 → 四极 → 化龙 → 仙台 → 大帝
```

#### 主角的独特路径
- 以 AI 技术为根基，融合修真传承
- 在星际联邦的职业体系中攀升
- 在古遗迹中觉醒上古血脉

### 三、星球观

#### 主要星球类型
1. **主星（地球/人类母星）**：AI 高度发达，社畜文化盛行
2. **修真星**：保留古法传承的封闭星球
3. **科技星**：纯粹科技发展，AI 治理
4. **资源星**：星际 mining 基地，底层劳工聚集地
5. **边境星**：人类文明边缘，危险与机遇并存

### 四、故事观

#### 核心主题
- **个人成长**：从底层社畜到宇宙巅峰
- **文明责任**：个人命运与人类文明绑定的使命感
- **人性光辉**：在绝境中依然保持善良和正义
- **探索精神**：对未知宇宙的好奇与敬畏

#### 叙事节奏
- 每 3-5 章一个小高潮
- 每 20-30 章一个大高潮
- 每卷一个完整的起承转合

## 人物设定

### 主角：林默

#### 基本信息
- **年龄**：32 岁
- **职业**：AI 算法工程师（被裁员边缘）
- **出身**：普通家庭，小镇做题家
- **性格特点**：
  - 坚韧不拔：再难也不放弃
  - 正直善良：有底线，不坑人
  - 隐忍内敛：不善于表达情绪
  - 聪明机智：善于用智慧解决问题
  - 不服输：越挫越勇的性格
  - **重情重义**：对朋友爱人全心全意，成功后不忘本，愿与亲友共富贵
  - **有血有肉**：不是冷血修炼机器，渴望巅峰时也有爱人和朋友相伴

#### 核心信念
- "就算站到宇宙巅峰，我也不想孤身一人"
- "修炼不是为了抛弃亲人爱人，而是为了更好地守护他们"
- "如果成仙成佛的代价是断情绝爱，那这仙佛不成也罢"

#### 外貌特征
- 身高 178cm，偏瘦
- 黑眼圈明显（长期加班）
- 眼神坚毅（核心特征）
- 穿着简单（程序员标配）

#### 成长弧光
```
被裁员 → 获得奇遇 → 进入星际联邦 → 发现上古秘密 
→ 卷入星际纷争 → 成长为传奇 → 证道宇宙巅峰
→ 携亲友爱人共登巅峰，成就完美人生
```

#### 金手指/奇遇
- **上古 AI 核心**：融合修真与科技的奇异存在
- **功能**：
  - 推演功法（科技侧）
  - 解析遗迹（修真侧）
  - 优化修炼（双轨融合）

### 重要配角

#### 1. 苏雨晴（女主）
- **身份**：星际联邦年轻军官，苏家嫡系传人
- **性格**：果敢、正义、有领导力，外冷内热
- **与主角关系**：从互相利用到生死与共，最终成为主角道侣
- **成长线**：家族联姻的牺牲品 → 独立自主的女强人 → 与主角并肩作战的伴侣
- **结局**：与主角一起证道，成为宇宙巅峰的帝后

#### 2. 王铁柱（死党/王胖子）
- **身份**：月球基地室友，地球修真世家王家旁系
- **性格**：乐观、仗义、话多，关键时刻靠得住
- **与主角关系**：患难与共的好兄弟，始终信任主角
- **作用**：提供情报、调节气氛、生死时刻挡刀
- **结局**：成为主角麾下星域总管，一方霸主

#### 3. 老陈（导师角色）
- **身份**：主角前公司技术总监，隐退大佬，上古传承者
- **性格**：表面散漫，实则深谋远虑，护短
- **与主角关系**：亦师亦父，主角的引路人
- **作用**：指点主角修炼，提供上古秘辛
- **结局**：为保护主角牺牲，后复活

#### 4. 李晓芸（红颜知己）
- **身份**：自由搏击冠军，孤儿出身
- **性格**：直爽、敢爱敢恨、不服输
- **与主角关系**：暗恋主角，但选择默默守护
- **结局**：成为主角麾下战神，终身未嫁

#### 5. 张子轩（智囊）
- **身份**：清华大学物理系博士，科技侧天才
- **性格**：理性、谨慎、书生气
- **与主角关系**：被主角折服，成为军师
- **结局**：帮助主角建立宇宙级科研体系

#### 6. 雷教练（亦师亦友）
- **身份**：月球基地总教练，职业者 5 级
- **性格**：豪爽、正直、惜才
- **与主角关系**：欣赏主角，暗中相助
- **结局**：成为主角麾下舰队指挥官

#### 7. 无名尊者（精神导师）
- **身份**：上古纪元最后一位大能者，'起源'原主人
- **性格**：大爱无疆，舍己为人
- **与主角关系**：虽已陨落，但精神指引主角
- **作用**：留下'起源'，布局万古

#### 8. 反派：赵天成
- **身份**：主角前公司高管，后投靠外星势力
- **性格**：自私、贪婪、不择手段
- **与主角关系**：多次陷害，最终对决
- **结局**：众叛亲离，被主角击败

#### 9. 反派：赵老爷子
- **身份**：赵家家主，职业者 7 级
- **性格**：老谋深算，护短，家族利益至上
- **与主角关系**：因主角让赵家蒙羞而结仇
- **结局**：赵家覆灭，被主角斩杀

## 创作注意事项

1. **人物塑造**：主角性格必须始终如一，成长要有铺垫
2. **场景描写**：每个场景都要有画面感，让读者身临其境
3. **节奏把控**：张弛有度，高潮后要有缓冲
4. **逻辑自洽**：设定不能前后矛盾
5. **情感共鸣**：社畜困境要真实，逆袭要爽快
6. **留白艺术**：不要把所有话说尽，给读者想象空间
7. **主角性格**：必须体现重情重义，不能成为冷血修炼机器
8. **感情线**：与苏雨晴的感情要循序渐进，不能突兀
9. **兄弟情**：王铁柱等配角要有自己的成长线，不能只是工具人
10. **反派塑造**：反派也要有血有肉，不能脸谱化
